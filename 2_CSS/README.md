## position

- 웹 문서 안에서 요소들을 자유롭게 배치하는 속성
- static : 문서의 흐림에 맞추어 요소를 배치(기본값)
- relative : 이전 요소에 연결해 배치하면서 위치를 지정하여 요소를 배치
- absolute : 원하는 위치를 지정하여 요소를 배치
- fixed : 지정한 위치에 고정하여 요소를 배치
- 속성값이 static을 제외한 나머지들은 좌표를 이용하여 요소의 위치를 조절할 수 있다.
